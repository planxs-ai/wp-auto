# AutoBlog 고객 시작 가이드

> 30DaysLiving AutoBlog — AI가 매일 블로그 글을 자동으로 써주는 서비스

---

## 시작 전 준비물

| 항목 | 설명 |
|------|------|
| WordPress 블로그 | 자체 호스팅 WordPress (wordpress.com 무료 플랜 불가) |
| 이메일 주소 | 회원가입 및 인증에 사용 |
| PC 또는 모바일 | 브라우저만 있으면 OK |

> **WordPress가 없다면?** 카페24, 클라우드웨이즈 등에서 월 5,000원대로 생성 가능합니다. 별도 안내 요청 시 도와드립니다.

## STEP 1. 워드프레스 앱 비밀번호 생성

AutoBlog이 글을 발행하려면 워드프레스 "앱 비밀번호"가 필요합니다.

1. 워드프레스 관리자 로그인 (`내블로그.com/wp-admin`)
2. 좌측 메뉴 → **사용자** → **프로필**
3. 페이지 맨 아래 **"앱 비밀번호(Application Passwords)"** 섹션
4. 새 앱 비밀번호 이름: `AutoBlog` 입력
5. **"새 앱 비밀번호 추가"** 클릭
6. 생성된 비밀번호를 **반드시 복사해서 메모** (다시 볼 수 없음!)

> **형식 예시:** `ABCD 1234 EFGH 5678 IJKL 9012` (공백 포함 그대로 사용)

---

## STEP 2. 회원가입

1. **https://30daysliving.com/login** 접속
2. **"회원가입"** 탭 클릭
3. 정보 입력:
   - **이름**: 대시보드에 표시될 닉네임
   - **이메일**: 인증 메일을 받을 주소
   - **비밀번호**: 6자 이상
4. **"회원가입"** 버튼 클릭
5. 이메일 수신함에서 **인증 메일 확인** → 링크 클릭

> 가입 즉시 **7일 무료 Premium 체험**이 시작됩니다!

---

## STEP 3. 초기 설정 (온보딩 5단계)

가입 후 자동으로 온보딩 화면이 나타납니다. 순서대로 진행하세요.

### 3-1. 워드프레스 연결

| 입력 항목 | 예시 |
|-----------|------|
| 사이트 URL | `https://myblog.com` |
| 사용자명 | 워드프레스 로그인 ID |
| 앱 비밀번호 | STEP 1에서 복사한 비밀번호 |

**"연결 테스트"** 버튼을 눌러 성공 확인 후 다음으로 진행합니다.

> **연결 실패 시 체크리스트:**
> - URL에 `https://` 포함했는지 확인
> - 앱 비밀번호를 일반 로그인 비밀번호와 혼동하지 않았는지 확인
> - 워드프레스 REST API가 활성화되어 있는지 확인 (대부분 기본 활성)

### 3-2. 카테고리 선택

블로그에 쓸 주제를 **최소 2개** 선택합니다.

**인기 조합 추천:**

| 조합 | 특징 |
|------|------|
| AI 도구 + 재테크 | 수익화 ROI 최고 |
| 스마트홈 + 가전 | 쿠팡 연동에 유리 |
| 건강 + 뷰티 | 텐핑 CPA 전환율 높음 |
| 정부지원 + 세금 | 검색량 안정적 |

> 🔒 표시가 있는 카테고리는 Premium 전용입니다.

### 3-3. 발행 스케줄 선택

| 옵션 | 시간 | 추천 대상 |
|------|------|-----------|
| 하루 2회 | 08:00, 18:00 | 초보자 |
| 하루 4회 | 07:00, 12:00, 17:00, 22:00 | 빠른 성장 원하는 분 |
| 평일만 1회 | 08:00 (월~금) | 보수적 운영 |
| 커스텀 | 자유 설정 | Premium 전용 |

> **추천:** 애드센스 승인 전이라면 **하루 4회**로 빠르게 글 수를 쌓으세요.

### 3-4. 블로그 단계 선택

현재 블로그 상태에 맞는 단계를 선택합니다.

| 단계 | 선택 기준 |
|------|-----------|
| **신규 블로그** | 글이 거의 없는 새 블로그 |
| **애드센스 준비 중** | 글은 있지만 아직 애드센스 미승인 |
| **애드센스 승인 완료** | 애드센스 수익 발생 중 |
| **수익화 단계** | 다채널 수익 (쿠팡+텐핑 등) 운영 중 |

> AI가 이 단계에 맞춰 글의 스타일과 제휴 링크 포함 여부를 자동 조절합니다.

### 3-5. 설정 확인

모든 설정을 검토하고 **"시작하기"** 를 누르면 대시보드로 이동합니다.

## STEP 4. 초기 블로그 세팅 (중요!)

대시보드 진입 후 **설정** 페이지에서 3가지 초기 작업을 실행합니다.

| 순서 | 버튼 | 역할 | 소요시간 |
|------|------|------|----------|
| 1 | 📌 **메뉴 설정** | 소개/개인정보처리방침/연락처 페이지 + 메뉴 자동 생성 | 1~2분 |
| 2 | 🎨 **CSS 주입** | 블로그 테마 스타일링 적용 | 1~2분 |
| 3 | 📝 **첫 발행 (3편)** | AI가 3편의 글을 자동 작성/발행 | 5~10분 |

> **3개 모두 반드시 실행하세요!** 애드센스 승인에 필요한 필수 페이지와 초기 콘텐츠가 생성됩니다.

## STEP 5. 일상 사용법

설정 완료 후에는 **AI가 스케줄에 맞춰 자동으로 글을 발행**합니다. 별도 작업이 필요 없습니다.

### 대시보드 확인 (홈)

- **오늘 발행 수**: 오늘 몇 편이 발행되었는지
- **이번 달 수익**: 현재까지 누적 수익
- **블로그 건강 점수**: 0~100점 (높을수록 좋음)
- **마일스톤**: 첫 글 → 10편 → 30편 → 애드센스 승인 → 첫 수익 달성

### 내 블로그 탭

- 발행된 글 목록 확인
- 글별 품질 점수 확인 (90점 이상 = 우수)
- 카테고리 분포 차트
- 글 제목 클릭 → 실제 블로그에서 확인

### 수익 탭

- 월별 수익 추이 그래프
- 채널별 수익 비교 (애드센스/쿠팡/텐핑)
- **수익화 로드맵**: 현재 단계 + 다음 단계 가이드

## 자주 묻는 질문 (FAQ)

### Q. 글을 직접 쓰지 않아도 되나요?
**네.** AI가 키워드 선정 → 글 작성 → 이미지 삽입 → 워드프레스 발행까지 전부 자동으로 합니다. 설정만 해두면 매일 스케줄에 맞춰 발행됩니다.

### Q. 애드센스 승인은 얼마나 걸리나요?
보통 **20편 이상 + 필수 페이지(소개/개인정보/연락처) 완비** 후 신청하면 1~2주 내 승인됩니다. 초기 세팅에서 메뉴 설정을 실행하면 필수 페이지가 자동 생성됩니다.

### Q. 글 품질이 걱정됩니다.
Premium 이상 플랜에서는 **골든 모드**가 활성화되어 가짜 통계나 날조된 정보가 포함되지 않습니다. 품질 점수 90점 이상이 대부분입니다.

### Q. 여러 블로그를 운영할 수 있나요?
- Standard: 1개
- Premium: 최대 3개
- MaMa: 무제한

### Q. 중간에 카테고리를 바꿀 수 있나요?
**네.** 설정 → 카테고리 선택에서 언제든 변경 가능합니다.

### Q. 워드프레스 연결이 안 됩니다.
1. URL에 `https://` 포함 확인
2. **앱 비밀번호**를 사용했는지 확인 (일반 비밀번호 아님!)
3. 워드프레스 버전 5.6 이상인지 확인
4. 보안 플러그인(Wordfence 등)이 REST API를 차단하고 있지 않은지 확인

### Q. 발행된 글을 수정할 수 있나요?
**네.** 워드프레스 관리자 페이지에서 직접 수정 가능합니다. 대시보드 "내 블로그" 탭에서 글 제목 옆 링크(↗)를 클릭하면 해당 글로 이동합니다.
